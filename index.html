Sobre Mí

Hola soy Ezequiel tengo 27 años, un desarrollador de software con 4 años de experiencia. Mi especialidad es el desarrollo de BackEnd, donde he profundizado en el uso de Node.js. Tengo experiencia en la gestión de aplicaciones en la nube con Amazon Web Services, lo que me permite garantizar la escalabilidad de las aplicaciones que desarrollo. Aunque mi fortaleza no es el desarrollo FrontEnd, he implementado React.js para la interfaz de usuario y React Native para el desarrollo de aplicaciones universales ‘bridge’. Estoy en constante crecimiento y aprendizaje, y cada día elijo la tecnología porque es lo que realmente me apasiona.

Proyecto Destacado: Desarrollo de una aplicación TODO robusta y dinámica del lado del cliente utilizando React Native y Expo. Se definieron rutas con sus respectivos endpoints y se desarrolló lógica para realizar peticiones desde el cliente. La implementación de CORS facilitó la comunicación efectiva entre el front y el back. Este proyecto destaca por la integración experta de tecnologías en ambas capas, cliente y servidor, para ofrecer una aplicación sólida y completa.
